TanmayFixer

Windows & Linux Recovery Utility

Repair. Backup. Recover.

TanmayFixer is a recovery toolkit made for fixing broken Windows and Linux systems. It includes backup tools, driver export, USB recovery preparation, and future Linux support.

GitHub Repository Download

Features

Backup User Data

Backup Desktop, Documents, Downloads, Pictures, Videos, and Music safely.

Driver Export

Export Windows drivers for recovery and reinstall purposes.

USB Recovery Preparation

Prepare USB drives with recovery files, backups, and drivers.

EFI Protection

Dangerous EFI and recovery partitions are hidden for safety.

Windows Repair

Run Windows repair tools and recovery operations easily.

Linux Support (Coming Soon)

Future versions will support Linux recovery, GRUB repair, and Linux backups.
